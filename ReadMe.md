#  About Me:
I’m a 3rd-year B.Tech (ECE) student at IIIT Una passionate about building scalable full-stack applications enhanced with Machine Learning and Generative AI. I specialize in combining robust backend systems with intelligent AI capabilities to create real-world, impactful products.

My goal is to bridge the gap between traditional web development and next-generation AI systems, turning ideas into applications that are not just functional, but also smart and adaptive.

# What I Do
⚙️ Build end-to-end full-stack applications using the MERN stack
🤖 Integrate LLMs and AI models into web apps for automation and intelligence
🔗 Design AI-powered workflows using LangChain & LangGraph (Agentic AI)
